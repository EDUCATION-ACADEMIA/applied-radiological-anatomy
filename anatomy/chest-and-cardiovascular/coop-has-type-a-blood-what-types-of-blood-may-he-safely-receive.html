Coop has Type A-blood. What types of blood may he safely receive?

(A) ** Type A-Type O-

(B) Type A-Type A+Type O-Type O+

(C) Only Type A-

(D) Only Type O-

EXPLANATIONS BELOW

Concept note-1: -If you have type A blood, you can only receive types A and O blood. If you have type B blood, you can only receive types B and O blood. If you have type AB blood, you can receive types A, B, AB, and O blood. If you have type O blood, you can only receive type O blood.

Concept note-2: -Type O blood does not cause an immune response when it is given to people with type A, type B, or type AB blood. This is why type O blood cells can be given to people of any blood type. People with type O blood are called universal donors.

However, they cannot receive blood from a donor with blood group B or AB (which both have the B antigen).
