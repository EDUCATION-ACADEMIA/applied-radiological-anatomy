Which of the following is an accurate analogy/comparison for the nervous system system?

(A) The nervous system like the United Sates Postal Service that deliver important letters and packages across all the highways in the country

(B) The nervous system is like the frame of a car that supports and protects it to allow for mobility.

(C) The nervous system is like a recycling center that sorts the reusable trash from the rest of the waste.

(D) ** The nervous system is like air traffic control that processes airway information in order to send messages to plane on when and where to land.

EXPLANATIONS BELOW

Concept note-1: -As a straightforward analogy, consider the nervous system to be like a giant tele-phone system. What makes memories, thoughts, vision, speech, hearing, and pain different is that each system has a different area code and telephone number.

Concept note-2: -The nervous system and endocrine system are similar because they both allow for communication throughout the body. The nervous system uses electrical signals and chemical messengers called neurotransmitters to communicate short distances between neurons.

Concept note-3: -The neuron is the cell that makes up the nervous system. The nerve is a bundle of neurons. The neuron is responsible for generating nerve impulses in the form of chemical and electrical signals whereas nerves provide the environment for conducting the nerve impulses.
